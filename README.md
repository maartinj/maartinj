## MedScan 💊 ✨
[App Store link](https://apps.apple.com/app/apple-store/id6747271665)

<img width="1920" height="1080" alt="medScanPromo" src="https://github.com/user-attachments/assets/fc0b50fc-f82d-429d-b638-28f79ec3d0c4" />

MedScan is an iOS app for identifying medications. Take a photo of a medication's packaging, and the image recognition system suggests the medication. After scanning, it displays detailed information about the medication - name, dosage and potential side effects.
<br><br>
Tech: SwiftUI, GPT-4 Vision, MVVM.
